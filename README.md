# 121-Selfbot
Louding micing selfbot | Can make you loudest on cord.

# How to use?
1. Clone the respository by running:
```bash
git clone https://github.com/NotNishanXD/121-Selfbot.git
```
2. Run:
```bash
cd 121-Selfbot
```
3. Run:
```bash
npm install
```
4. Run the selfbot and you're done!:
```bash
node index.js
```
